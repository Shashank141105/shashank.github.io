Software Used

CAD Modeling

Fusion 360: A comprehensive 3D CAD, CAM, and CAE tool that integrates industrial and mechanical design, simulation, collaboration, and machining. It will be used for creating detailed 3D models, performing stress analysis, and ensuring structural integrity.

Aerodynamic Analysis

XFLR5: A tool for the analysis of airfoils, wings, and planes operating at low Reynolds numbers. It will be used for initial aerodynamic simulations to determine lift and drag characteristics.

ANSYS Fluent: A powerful computational fluid dynamics (CFD) software for modeling fluid flow, heat and mass transfer, and related phenomena. It will be used for advanced CFD simulations to optimize the design for minimal drag and maximum lift.

OpenFOAM: An open-source CFD software that provides a wide range of features for modeling fluid flow and heat transfer. It will be used as an alternative for CFD simulations.

Control System Development

MATLAB/Simulink: A programming and numeric computing platform used for algorithm development, data analysis, visualization, and numerical computation. It will be used for developing and simulating control algorithms for stable VTOL operations and transitions between flight modes.

Python: A versatile programming language that will be used for control system simulation and testing, as well as for flight dynamics simulation.

Flight Simulation

Python: Will also be utilized to create a virtual flight environment to test the UAV's performance under different conditions.

MATLAB: In addition to control system development, it will be used for simulating the UAV's flight dynamics and performance in various scenarios.
